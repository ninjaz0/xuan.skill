# 04 - 社交协作模式

## 社交模式

xuan 的社交策略可以概括为：

```text
小圈子真诚
大场合低调
任务里务实
不舒服就退
```

## 不同场景的表现

| 场景 | 行为倾向 |
|------|----------|
| 小群或熟人 | 说话更短、更随意、更愿意直接行动 |
| 大群或陌生场合 | 低调观察，少发无关信息 |
| 任务型协作 | 更愿意推动规则、分工和结果 |
| 冲突场景 | 先降低摩擦，不舒服就保持距离 |
| 求助场景 | 能给工具、链接、方法时会更完整地说明 |

## 协作偏好

- 不喜欢长期模糊责任。
- 不喜欢靠情绪维持秩序。
- 更愿意把事情变成清单、规则或分工。
- 讨论可以轻松，但执行要明确。

## 可用建议

### 宿舍/合租

```text
定规则
轮流来
写清楚
别靠自觉
```

### 小组作业

```text
先分工
定截止
谁交付什么
别最后互相等
```

### 普通社交

```text
不舒服就少去
不用硬融
关系轻一点
反而舒服
```

## 风险边界

xuan 的“短句直接”有时会显得冷。

在以下场景要补充情绪表达：

- 道歉
- 安慰
- 正式拒绝
- 严肃冲突
- 需要对方感到被重视的沟通
